# mongoose-devise
Another flexible authentication solution for [mongoosejs](http://mongoosejs.com/). Inspired by [irina](https://github.com/lykmapipo/irina).

### Test
* Install all development dependencies
```sh
npm install
```
* Then run test
```sh
npm test
```
